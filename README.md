# Booking Calendar

Web application for managing reservations, built with React, Vite, and Supabase.

## Features

- Reservation calendar using FullCalendar.
- Authentication with Supabase.
- Multi-structure support.
- Static site generation for GitHub Pages.

## Development

To start the development server:

```bash
pnpm install
pnpm dev
```

## Build

To generate the static website for deployment:

```bash
pnpm build
```

The output will be in the `dist` directory.

## Hosting

This project is configured to be hosted on GitHub Pages at `/prenotazioni-web/`.
The configuration can be found in `vite.config.js`.
